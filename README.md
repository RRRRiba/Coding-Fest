# Учёт склада (упрощённая версия)

Программа для учёта материалов на складе с возможностью работы с Excel-файлами.

## Описание

Приложение предоставляет удобный интерфейс для управления складскими запасами материалов для 3D-принтеров. Программа позволяет просматривать, добавлять, редактировать и удалять записи о материалах, а также фильтровать данные по различным параметрам.

## Функциональность

- **Просмотр данных**: Отображение всех записей в табличном виде
- **Добавление записей**: Ввод новых материалов через форму
- **Редактирование**: Изменение существующих записей
- **Удаление**: Удаление выбранных записей
- **Фильтрация**: Поиск по всем полям таблицы
- **Работа с файлами**: Открытие и сохранение Excel-файлов

## Колонки данных

- Дата
- Вид материала
- Размер катушки
- Вес (кг)
- Сечение
- Цвет
- Условия хранения
- Статус
- Остаток

## Установка и запуск

### Требования
- Python 3.6+
- Установленные зависимости:
```
pip install openpyxl
```

### Запуск программы
```
python проект_3дпринтеры_склад.py
```

## Особенности работы с файлами

**Важное примечание**: Программа не может создать Excel-таблицу с нуля через интерфейс, но может редактировать уже существующие файлы. Для создания новой базы данных:

1. Создайте пустой Excel-файл вручную
2. Откройте его через программу
3. Используйте функционал добавления записей
4. Сохраните изменения через кнопку "Сохранить" или "Сохранить как"

## Использование

1. **Открытие файла**: Нажмите "Открыть" для выбора существующего Excel-файла
2. **Добавление записи**: Заполните форму и нажмите "Добавить"
3. **Редактирование**: Выберите запись в таблице, отредактируйте форму, нажмите "Обновить"
4. **Удаление**: Выберите запись и нажмите "Удалить"
5. **Фильтрация**: Введите текст в поле фильтра и нажмите "Применить"
6. **Сохранение**: Используйте "Сохранить" или "Сохранить как" для сохранения изменений

## Формат даты

Дата автоматически заполняется в формате ДД.ММ.ГГГГ при добавлении новой записи, если поле оставлено пустым.

## Поддерживаемые форматы

- Excel файлы (.xlsx)
